# Tech Prime Internship Dart Exercises

This repository contains Dart practice exercises completed during the Tech Prime internship. The examples cover Dart fundamentals, collections, object-oriented programming, inheritance, constructors, and asynchronous programming.

## Project Structure

- `dayFirst`: Variables, functions, conditions, operators, and basic calculations
- `daySecond`: Functions, lists, closures, and returning functions
- `dayThird`: Sets and set methods
- `dayFour`: Classes, objects, maps, instance methods, and exception handling
- `dayFive`: Constructors, named constructors, private members, and `super`
- `daySix`: Constant, redirecting, getter/setter, and inheritance examples
- `daySeven`: Abstract classes, interfaces, method overriding, enums, and static members
- `dayEight`: A console-based Student Management System

## Requirements

- [Dart SDK](https://dart.dev/get-dart) installed and available on your PATH

## Run an Exercise

From the repository root, run any Dart file with:

```powershell
dart run dayFirst/task_Check_Number.dart
```

For example, to run the Student Management System:

```powershell
dart run dayEight/project_SMS.dart
```

The Student Management System demonstrates:

- Classes, inheritance, and an abstract interface
- Student validation for marks from 0 to 100
- List-based storage and map-based RegNo lookup
- Exception handling for invalid input
- An asynchronous server-sync simulation
